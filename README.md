# Storage-billing-system project

## Project setup

### Goto Project Directory

```
cd storage-billing-system
```

### Install Project Dependency
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
